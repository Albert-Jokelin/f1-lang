Fun parser that lets you write C style code with F1-themed keywords. Here's a list of keywords used:
- "through_goes" = "return"
- "lights_off" = "main"
- "POLE" = "false"
- "DNF" = "true"
- "drs" = "while"
- "push" = "if"
- "maintain" = "else"
- "pit"= "break"
- "teamRadio" = "printf"
